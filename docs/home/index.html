EP16: Missing Values in Clinical Research

Multiple Imputation


[This website is currently under construction]

This website contains slides and materials for the Multiple Imputation part of NIHES course Missing Values in Clinical Research (EP16).

Practicals are available as interactive learnr tutorials (links will follow) as well as in a static html version.

The practicals require R and JAGS to be installed and use the following R packages: (JAGS (and rjags) are only necessary for the parts of the practical that use the JointAI package.)

  • mice (version 3.3.0)
  • miceadds (version 3.0-16)
  • mitools (version 2.3)
  • JointAI (version 0.4.0)
  • rjags (version 4-8)
  • smcfcs (version >= 1.3.1)
  • jomo (version >= 2.6-7)

For JointAI it is suggested to use the most recent version available on GitHub

devtools::install_github("NErler/JointAI")
